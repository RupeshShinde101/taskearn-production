Admin Panel Moved

Use WorkmateAdmin App

The web-based admin panel has been consolidated into the WorkmateAdmin desktop app for security. All features are available there.

  1. Open WorkmateAdmin folder on your Desktop
  2. Double-click Start Admin Panel.bat
  3. Open http://localhost:5000 in your browser
  4. Log in with your admin credentials

For security reasons the admin panel is no longer publicly accessible via the website.
